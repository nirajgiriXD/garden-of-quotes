# Garden of Quotes

Welcome to the `Garden of Quotes`.

## Details

- The data is in `public/quotes.json`.
- A single data consists of three key-value pair:
  - `"quote"`: `""`
  - `"author"`: `""`
  - `"tag"`: `[]`

## Format

```JSON
{
  "data": [
    {
      "quote": "",
      "author": "",
      "tag": []
    }
  ]
}
```
